Roll a Ball



Created this game using Unity. The game starts with 12 cubes and the user collects the cubes by rolling over them with the ball. The game is over when all 12 cubes have been collected.

BattleShip


We created Javascript using Javascript and JQuery. Their is a 10 by 10 board on the screen and the user clicks on each box to see if they have hit or missed a ship. Their are two ships (1 horizontal and 1 vertical) on the board and they each have a length of 5 boxes. The user gets 25 clicks to reveal both ships.
